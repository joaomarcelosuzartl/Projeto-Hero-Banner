# 🚀 Projeto Hero Banner

Landing page com seção **Hero Banner** (topo de impacto) construída com **HTML + CSS + JavaScript**, ideal para treinar layout, responsividade e organização de assets (imagens/estilos/scripts).
> Autor: João Marcelo  
> Repositório: Projeto Hero Banner
## ✨ Preview (como rodar)
Você pode abrir o projeto de duas formas:
### Opção A — Abrir direto
1. Baixe/clone o repositório
2. Abra o arquivo `index.html` no navegador
### Opção B — Rodar com Live Server (recomendado)
Se você usa VS Code:
1. Instale a extensão **Live Server**
2. Clique com o botão direito em `index.html` → **Open with Live Server**
## 🧩 Funcionalidades
- Seção **Hero** com destaque visual (texto + imagem)
- Estrutura pronta para navegação (menu/topo)
- Área de conteúdo com imagens e seções informativas
- Organização por pastas (assets separados do HTML)
## 🛠️ Tecnologias
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
